# QuranMind

An AI-powered Quranic study web application built as part of the 
LJMU BSc Computer Science final year project (6200COMP).

## Running the app
1. Open QuranMind_App.html in a browser
2. Enter your Anthropic API key when prompted
3. Browse Surahs and ask questions about verses

## Tech stack
- HTML, CSS, JavaScript 
- Al-Quran Cloud API for Quranic data
- Anthropic Claude API for AI explanations
